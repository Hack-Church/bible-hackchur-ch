Below is a list of resources compiled by the Hack Church community:

## Pentesting Distros
Pentesting distros are VMs that come pre-configured with common tools:

- [Kali Linux](https://www.kali.org/downloads/) by [Offensive Security](https://www.offensive-security.com/)
- [Commando VM](https://github.com/fireeye/commando-vm) by [FireEye](https://www.fireeye.com/)

## Wargames
- [Hack The Box](https://hackthebox.eu) - linux, windows, web, reverse engineering
- [Microcorruption](https://microcorruption.com/login) - reverse engineering, assembly, gcc
- [Over The Wire](https://overthewire.org/wargames/) - bash, cryptography, linux, reverse engineering, web
- [Smash The Stack](http://smashthestack.org/wargames.html)
